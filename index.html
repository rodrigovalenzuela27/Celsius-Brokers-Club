Plataforma Celsius. Tres vistas, una arquitectura.

§04 · Notas

¿Cómo navego el prototipo?

Cada vista (Broker, Cliente, Admin) abre como una app independiente. Dentro de cada una hay un sidebar (Broker, Admin) o top-nav (Cliente) que permite saltar entre todas las secciones funcionales. Los datos mostrados son representativos pero no en vivo — el prototipo demuestra interacciones, no operación real.

Las tres vistas están interconectadas vía una barra superior negra que permite saltar entre ellas sin perder contexto.

¿Qué representa la marca?

Los prototipos aplican la paleta de Celsius (charcoal #383838, cyan #009DEA, texto claro #F5F5F5) extraída del sitio institucional 0celsius.mx. Tipografía: Inter Tight como sans primaria, JetBrains Mono para metadata, Source Serif 4 italic para acentos editoriales.

El prototipo es la traducción operativa de la marca institucional al producto digital.
